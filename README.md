# Dataset

This is a reasoning dataset created using [Claude Opus 4.6](https://openrouter.ai/anthropic/claude-opus-4.6) with reasoning effort set to Adaptive.

The dataset is meant for creating distilled versions of [Claude Opus 4.6](https://openrouter.ai/anthropic/claude-opus-4.6) by fine-tuning already existing open-source LLMs.

This dataset includes an addition to our recently enhanced set of prompts to cover creative writing and multilingual creative writing.

## Files

- Input dataset: [Programming language 2000x](./dataset/programming-language-source-2000x.jsonl)
- Output dataset: [Claude Opus 4.6](./dataset/claude-opus-4.6-high-reasoning-700x.jsonl)
